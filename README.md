# CI/CD Pipeline Projects

A collection of projects demonstrating complete CI/CD pipelines built using **Docker**, **Docker Compose**, **Terraform**, **Ansible**, **AWS EC2**, and **GitHub Actions**.

Each project within this repository contains its own dedicated README with setup instructions, architecture details, and deployment workflows. This central repository serves as an index for showcasing and organizing all DevOps automation projects.

## Tools & Technologies

- **Docker** – Containerization of applications
- **Docker Compose** – Multi-container orchestration
- **Terraform** – Infrastructure as Code for provisioning AWS resources
- **Ansible** – Configuration management and remote server automation
- **AWS EC2** – Cloud compute for deployments
- **GitHub Actions** – Continuous Integration and Deployment automation

## Purpose

This repository highlights hands-on implementations of modern CI/CD practices using real infrastructure automation tools.
